# Diseño

- **Canvas:** `colors.background.default` (`#ffffff`).
- **Cards / listas:** `colors.neutral.gray[100]` o `colors.background.info` con borde `colors.border.main`.
- **Texto:** `colors.text.primary` / `secondary` / `tertiary`.
- **Marca / enlaces:** `colors.primary[500]`–`[700]`.
- **Éxito (montos secundarios, badges):** `colors.success.*` donde aplique semánticamente.
- **CTA principal:** `colors.primary[500]` sólido; deshabilitado `colors.states.disabled.*`.
- **CTA WhatsApp:** `colors.success.main` sólido (sin gradiente).
- **Total / barra fuerte:** `colors.neutral.gray[900]` + `colors.text.inverse` para mantener jerarquía sin gradiente.
